# huawei_e3_e5
Tool to manage huawei hilink modems (E3 and E5 series)


## Features

- [x] Get modem status
- [ ] Change sim settings
- [x] Get SMS messages
- [x] Send SMS messages
- [x] Modem reboot
- [x] Dump all modem infos
